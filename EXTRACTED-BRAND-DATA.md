## 1. Color variables

### Core brand (theme CSS)

| Variable role   | Hex        | Usage |
|-----------------|------------|--------|
| **Primary**     | `#564298`  | Main CTA, in-page links, focus ring, active nav. Use sparingly. |
| **Primary light** | `#7B63B8` | Primary hover. |
| **Primary dark**  | `#3F2F70` | Primary active; optional hero headline. |
| **Secondary**     | `#a8ddd4` | Cards, section accents, secondary buttons. Main warmth. |
| **Secondary light** | `#C5EAE4` | Soft backgrounds, highlights. |
| **Secondary dark**  | `#85C4B8` | Secondary hover; success states. |
| **Accent**         | `#9b8fb5` | Bullets, small decorative elements, tags. Not for body text. |
| **Background**     | `#ffffff` | Cards, panels, hero content. |
| **Background warm**| `#f6f5ef` | Page/section background (cream). |
| **Text**           | `#2e2e2e` | Body and headings. |
| **Text light**     | `#666666` | Secondary copy, captions. |
| **Border**         | `#E0E0E0` | Dividers, inputs, cards. |

### Semantic

| Role            | Hex        |
|-----------------|------------|
| **Success**     | `#85C4B8`  |
| **Warning**     | `#D4A843`  |
| **Urgent/Error**| `#C64B4B`  |
| **Info**        | `#2563eb`  |

**Contrast rule:** Purple for one primary CTA per view; teal and cream carry layout. Text contrast ≥4.5:1 (WCAG 2.1 AA).
